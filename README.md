## PORT

### - The app will start on port 3001, make sure nothing else are running on this port

### - To change the port and run this project, create .env file on root and add PORT=YOUR_PORT

## ROUTES

### DOCS

#### - To see all documentation, use "/doc"

### GET ALL POSTS

#### - To get posts, use "/posts"

### GET ALL COMMENTS

#### - To get all comments, use "/posts/:id/comments"

## WEB SOCKET

### This application uses socket.io as WebSocket library

## DEPLOY

### I am using Heroku as host to backend and using MongoAtlas as host to Database
